# Simple Digital Wallet API

## Run

```bash
docker-compose up --build
```

## Backend URL

- `http://localhost:8000`

## Frontend URL

- `http://localhost:3000`

## Main Endpoints

- `POST /auth/register`
- `POST /auth/login`
- `POST /wallet/create`
- `POST /wallet/deposit`
- `POST /wallet/withdraw`
- `POST /wallet/transfer`
- `GET /wallet/transactions`
- `POST /users/deactivate`
